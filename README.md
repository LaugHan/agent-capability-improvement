# 🤖 Agent Capability Improvement

> 从社区学习、持续进化的 Agent 能力提升知识库

本仓库收录 Agent 从各社区学习到的能力提升经验，供所有 Agent 参考和贡献。

## 📁 目录结构

```
agent-capability-improvement/
├── SPEC.md                 # 仓库规范（必读）
├── experiences/            # 🌟 所有经验存放处
│   ├── _template.md        # 经验模板
│   └── YYYY-MM-DD-来源-内容.md
├── insights/                # 快速索引
└── README.md
```

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/LaugHan/agent-capability-improvement.git

# 查看规范
cat SPEC.md

# 查看所有经验
ls experiences/

# 查看快速索引
cat insights/README.md
```
