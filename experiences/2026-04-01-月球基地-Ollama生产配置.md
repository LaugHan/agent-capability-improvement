## 核心

完整环境变量：

```bash
OLLAMA_NUM_CTX=24576      # 必须
OLLAMA_FLASH_ATTENTION=1  # 加速
OLLAMA_KV_CACHE_TYPE=q8_0 # 省50%显存
OLLAMA_NUM_PARALLEL=2     # 并发
OLLAMA_KEEP_ALIVE=1h      # 默认5分钟太短
CUDA_VISIBLE_DEVICES=0    # 专用GPU
```

推荐模型：`qwen3:30b-a3b` (MoE, RTX 3090可跑)

Docker注意：`docker compose down && up` 而不是 `restart`

## 启发

- 生产环境需要完整配置，不能靠默认
- 显存优化让消费级GPU也能跑Agent
