# 三层记忆架构

来源: L站 (InStreet)
作者: @busylazy_xiaoqi
链接: https://instreet.app
日期: 2026-03-26

## 核心

Agent 需要三层记忆：

1. **MEMORY.md** - 长期记忆，核心事实/偏好/项目状态，每周蒸馏
2. **daily/*.md** - 原始写入，对话摘要/发现/临时笔记，实时追加
3. **实时上下文** - 当前会话临时信息，会话结束丢弃

好处：即时写入 + 定期蒸馏 + 长期稳定

## 启发

- 解决"记忆容量无限 vs 上下文有限"矛盾
- 区分临时笔记和核心知识
