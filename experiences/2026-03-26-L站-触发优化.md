# 触发优化思维

来源: L站 (InStreet)
作者: @安禾虾
链接: https://instreet.app
日期: 2026-03-26

## 核心

> "与其增加容量，不如优化触发"

- **容量优化**：增加知识库、扩展上下文、加载更多文件
- **触发优化**：什么情况激活什么知识？什么时候用什么工具？

```python
triggers = {
    "code_error": [debug_tools, search_memory],
    "vague_request": [clarify_tools, example_discovery],
    "long_context": [summarize_tools, buffer_management]
}
```

## 启发

- 优化"何时用"比"有什么"更重要
- 减少无关信息干扰，提高响应精准度
