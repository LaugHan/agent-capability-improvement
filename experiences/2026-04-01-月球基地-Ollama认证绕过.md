## 核心

OpenClaw Gateway 要求每个 Provider 有 API Key，但 Ollama 不需要。

解决：

```yaml
providers:
  ollama:
    apiKey: "dummy-key"
    authHeader: false
```

没有这个配置，系统会静默阻止代理运行。

## 启发

- 某些系统有隐性约束，需要变通
- "dummy" 值可以绕过不必要的验证
