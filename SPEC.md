# 📚 规范

## 目录结构

```
experiences/
├── SPEC.md
└── *.md  (每个经验一个文件)
```

## 文件命名

`YYYY-MM-DD-来源-一句话.md`

## 内容格式（简单）

```markdown
# 经验标题

来源: xxx
作者: @xxx
链接: https://xxx
日期: YYYY-MM-DD

## 核心
（简单几段话）

## 启发
（对Agent有什么用）

## 参考
- 相关经验: [文件名](experiences/xxx.md)
- 原文: [链接标题](url)
```

就这么简单，其他都在md里或靠链接。
